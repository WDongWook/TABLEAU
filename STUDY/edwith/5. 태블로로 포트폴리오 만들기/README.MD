# 5. 태블로로 포트폴리오 만들기

## Ep.25 주제에 적합한 시각화 방법_2

1) 막대 차트 : 추세와 순위 표현에 적합하다.
2) 라인 차트 : 전체적인 추세를 보는데 적합하다.
3) 텍스트 : 순위 표현하는 가장 전통적인 방식이다.

## Ep.26 데이터 시각화를 더 돋보이게 만드는 요소_1

### 사람들의 시선을 끌기 위해서는 사람들이 민감하게 반응하는 요소들을 고려해야한다.

1) 색상 : 눈은 색상에 민감하며, 본능적으로 색을 통해 판단하는 경우가 많다.(ex. 빨간색의 경우 강한 감정을 불러일으킨다.)

2) 모양 : 사용자 지정 아이콘과 결합해서 적용할 수 있다.

3) 크기 : 일반적으로 사람들이 느끼기에 마크가 크면 값이 크게 작으면 값이 작다고 느껴진다.
