# 3. 태블로 중급 탐험


## Ep. 11 대시보드 만들기

- 대시보드 : 복수개의 워크시트로 구성되고, 시트 간 상호작용이 발생한다. 데이터를 발견하고 탐색한 후 그 과정에서 인사이트를 구하는 과정이다.
### 대시보드의 3가지 사항
- 1. 누가 볼 것인지 결정한다. -> 대시보드의 경우는 자기만족을 위해 구현하는 것이 아닌 다른 사람들이 보고 이해를 돕기 위해 만드는 것이다.
- 2. 보는 사람이 어떤 기기로 보는 것인지 중요하다. -> 보는 사람이 어느 장소에서 어떤 기기를 가지고 보는 것인지도 중요한 요소 중에 하나이다.
- 3. 보는 사람의 시선을 바로 잡을 수 있게 하는 것도 중요하다. -> 사람들의 시선은 주로 좌측 상단부터 내려오므로 중요하고 꼭 표현하고 싶은 것들이 있으면 좌측 상단에 두어 사람들의 시선을 사로 잡을 수 있다.
